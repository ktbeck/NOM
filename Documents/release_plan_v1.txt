----- HIGH LEVEL GOALS -----
- Have all functionalities integrated into a website.
- Be able to sell dining hall guest passes.
- Be able to buy dining hall guest passes.
- Be able to able to match buyers and sellers at a specific time and location for exchange.
- Have all functionalities integrated on mobile platforms.


----- SPRINTS + USER STORIES -----
[Effort required: 1 - very easy, 5 - very difficult]
Sprint 1: Be able to sell dining hall guest passes
  - [2] As a seller, I want to list my passes on the site so other students can buy them from me.
  - [3] As a seller, I want to have an account dashboard so I can keep track of my available guest passes.
  - [1] As a buyer, I want to see information about available passes so I can contact sellers and purchase them.

Sprint 2: Be able to buy dining hall guest passes
  - [5] As a seller, I want to receive payments through the app so I am guaranteed to be paid if the exchange happens.
  - [4] As a buyer, I want to give payments through the app so I can get a refund if the seller cancels.
  - [3] As a moderator, I want to have the ability to cancel or modify payouts so I handle disputes or cancellations.

Sprint 3: Be able to match buyers and sellers at a specific time and location.
  - [4]As a seller, I want to list when I go to the dining hall so I can easily swipe in buyers on my schedule.
  - [5]As a buyer, I want to see the location of all available sellers so I know immediately where I can go to get into a dining hall.
  - [3]As a user, I want to message the other user so I can set up meeting specifics.
